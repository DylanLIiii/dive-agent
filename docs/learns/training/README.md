## 定义

Agent Training 涵盖：
- **指令微调**：SFT (Supervised Fine-Tuning) 方法
- **强化学习**：RLHF、DPO 等对齐技术
- **多轮训练**：对话上下文中的训练策略
- **工具使用训练**：Function calling 能力培养
- **多 Agent 协作训练**：Multi-agent 场景训练
